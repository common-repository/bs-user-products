== Description ==

This plugin provides a field in Woocommerce product admin page to assign users with customer role. And when a user will be logged in and going through the shop pages, then only those products will be visible to them which are assigned to that user. You can assign multiple users to a single product.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Dashboard -> BS User Products settings page and enable the option.

== Frequently Asked Questions ==

= Is there a setting to enable/disable the products filter to show only assigned products? =

Yes, you can enable/disable plugin at any time from Dashboard -> BS User Products
